# fourier-propagation

## 1D API

The package includes a 1D transverse formulation E(x, z):

**Functions**
- `fresnel_propagate_fft_1d`, `fraunhofer_propagate_1d`, `angular_spectrum_propagate_1d`
- `rect_slit_1d`, `double_slit_1d`, `gaussian_1d`, `thin_lens_phase_1d`
- `plot_intensity_1d`
